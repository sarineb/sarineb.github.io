Predicting movement variability using machine learning with brain signals

Aim

Using machine learning to find out if the brain is responsible for the variability observed in movement details.

Background

  • Human movements vary in their details even when repeated exactly (goal-equivalent variability).
  • The mechanism of goal-equivalent variability in the nervous system is unknown.
  • Hypothesis: If goal-equivalent variability is produced at the brain level, the upcoming repetition class can be accurately predicted using EEG brain signals.

Experimental data

EEG and fingertip force data was obtained while healthy subjects performed a bimanual, two-finger force matching task where repetitions of the same task could be implemented using right>left or left>right forces. Our dataset consisted of EEG signals right before movement onset, as well as true force labels determined from the fingertip force sensors (right>left or left>right).

Design: mRova.
